# Bật GitHub Pages để mọi người vào link là thấy thiệp

Link hiện tại bị 404 vì **chưa bật Pages**. Làm đúng 4 bước sau:

---

## Bước 1: Vào repo trên GitHub

Mở: **https://github.com/buihonganhknight2901-lab/Birthday_MinhAnh**

(Đăng nhập GitHub nếu chưa.)

---

## Bước 2: Vào Settings

- Trên trang repo, bấm tab **Settings** (cùng hàng với Code, Issues, Pull requests…).

---

## Bước 3: Bật Pages

- Trong menu bên trái, tìm và bấm **Pages** (mục **Code and automation**).
- Phần **Build and deployment**:
  - **Source**: chọn **Deploy from a branch** (không chọn GitHub Actions).
  - **Branch**: chọn **main**, ô bên cạnh chọn **/ (root)**.
- Bấm **Save**.

---

## Bước 4: Đợi 1–2 phút

- Trên cùng trang **Pages** sẽ có dòng: *“Your site is live at https://buihonganhknight2901-lab.github.io/Birthday_MinhAnh/”*.
- Đợi 1–2 phút rồi mở link đó (hoặc F5 lại): thiệp sẽ hiện, mọi người vào link này đều thấy thiệp.

---

**Nếu vẫn 404:** Kiểm tra trong repo có file **index.html** ngay ở thư mục gốc (không nằm trong folder con). Nếu có → đợi thêm vài phút hoặc thử mở link bằng trình ẩn danh.
